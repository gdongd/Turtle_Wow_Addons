# GD_KeySaver - 魔兽世界乌龟服1.181 按键配置保存插件
一键保存/还原动作条、按键绑定、宏配置，适配乌龟服1.181版本

## 安装方法
1.  下载最新版ZIP：[GD_KeySaver v1.0.1](sslocal://flow/file_open?url=https%3A%2F%2Fgithub.com%2Fgdongd%2FTurtle_Wow_Addons%2Freleases%2Flatest&flow_extra=eyJsaW5rX3R5cGUiOiJjb2RlX2ludGVycHJldGVyIn0=)
2.  解压ZIP，把`GD_KeySaver`文件夹放到`魔兽世界\Interface\AddOns\`
3.  重启游戏，在插件列表勾选「GDKS乌龟服按键配置保存插件」

## 使用命令
- `/gdks`：打开配置管理面板

## 版本适配
- 仅支持：**魔兽世界乌龟服 1.181 (Interface 11200)**
- 不兼容正式服/其他怀旧服
